# My Waybar configuration

My personal configuration for Waybar. I basically just copied [mechabar](https://github.com/sejjy/mechabar) and changed a bunch of stuff to suit me better.

## Installation
clone the repo to ~/.config/waybar
``` git clone https://github.com/larkin/waybar ~/.config/waybar```

run `install.sh` to install the required dependencies
```~/.config/waybar/install.sh```

thats all!
